# Free / copyright-free beat sources

Download beats you like, drop the files into **`public/beats/`**, and add
each filename to **`public/beats/manifest.json`**, e.g.:

```json
["gfunk-90bpm.mp3", "boom-bap-88.mp3"]
```

They then appear (looping) in Beat Engine → My loops.

Note: almost all downloadable rap beats are **4/4** — for 3/4, 6/8, 12/8,
7/8 practice, the synth engine is the tool.

## Sources

- **Pixabay Music** (pixabay.com/music) — free, **no attribution
  required**. The cleanest option; filter by "hip hop", check BPM tags.
- **Free Music Archive** (freemusicarchive.org) — Creative Commons;
  license varies **per track**, check each one.
- **Chosic** (chosic.com/free-music/hip-hop) — royalty-free; **some
  tracks require attribution**, read each track's terms.
- **ShadowCast Studios** — CC BY-SA 4.0 (attribution + share-alike if you
  publish anything using them).
- **FreeBeats.io** — free downloads; **check the terms** per beat,
  especially before releasing anything recorded over them.

**Always check each track's individual license** — especially the moment
practice turns into something you publish. For practice-only use at home,
all of the above are fine.
